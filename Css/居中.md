# 居中

  方案一： Flex

  ```
    display: flex;
    justify-content: center;
    align-items: center;
  ```

  方案二：transform

  ```css
    #app {
      background: green;
      width: 200px;
      height: 400px;
      position: relative;
    }

    #child {
      background: red;
      width: 50px;
      height: 50px;

      position: absolute;
      top: 50%;
      left: 50%;
      transform: translate(-50%, -50%);
    }
  ```

  缺点：浏览器适配问题

  方案三：margin:0 auto;
