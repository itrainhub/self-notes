装饰者有四种
1. 类装饰者
2. 属性装饰者
3. 方法装饰者
4. 参数装饰者

优先级：
  p1: Parameter Decorator
  p2: Method Decorator
  p3. Accessor or Property Decorator
  p4. Class decorator
