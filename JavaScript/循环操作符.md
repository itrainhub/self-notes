# 2. for...in

```js
	for (const key in obj) {

	}

	return | break 会跳出循环, continue 则结束剩余代码执行，继续循环
```

# 3. for...of
    
```js
	for (const item of array) {

	}
```

return | break 会跳出循环, continue 则结束剩余代码执行，继续循环

如果需要拿到 index，ES6 支持 如下操作：

```js
	for (const [index, item] of array.entries()) {

	}
```
  对数组项进行迭代，item 为数组的某一项
